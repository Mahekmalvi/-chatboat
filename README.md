A web-based chatbot to recommend Muskbliss fragrances based on gender, age, occasion, and favorite notes.

## Features

- Simple, modern chat UI
- Smart suggestions
- Fetches product names, images, and links from [muskbliss.com](https://muskbliss.com)
- Mobile-friendly

## Setup

1. **Clone the repo** or copy these files.
2. **Install requirements**  
   ```
   pip install -r requirements.txt
   ```
3. **Run the Flask app**  
   ```
   python app.py
   ```
4. **Open in browser:**  
   Visit [http://localhost:5000](http://localhost:5000)

## Customization

- Edit `chatbot_logic.py` to change questions or add more notes/occasions.
- Edit `scraper.py` if Muskbliss site structure changes.
